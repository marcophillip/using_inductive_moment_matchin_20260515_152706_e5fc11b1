# Cloned Repositories

## Repo 1: IMM (Inductive Moment Matching)
- URL: https://github.com/lumalabs/imm
- Purpose: Official implementation of "Inductive Moment Matching" (Zhou et al., 2025).
- Location: `code/imm/`
- Key files:
  - `imm/models/`: Model architectures.
  - `imm/losses/`: MMD and other loss functions.
  - `scripts/`: Training and evaluation scripts.
- Notes: Uses PyTorch and focuses on generative modeling.

## Repo 2: D4RL
- URL: https://github.com/rail-berkeley/d4rl
- Purpose: Datasets for Deep Data-Driven Reinforcement Learning.
- Location: `code/d4rl/`
- Key files:
  - `d4rl/gym_mujoco/`: Environments and dataset URLs for Mujoco.
- Notes: Essential for loading the `half-cheetah-v2` dataset.
